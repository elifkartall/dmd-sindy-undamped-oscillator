## Overview

This project focuses on learning the dynamics of a nonlinear system (*Undamped Oscillator 2*) using data-driven approaches.

Two system identification methods are implemented and compared:

- **Dynamic Mode Decomposition (DMD)** – approximates the system using a best-fit linear model  
- **Sparse Identification of Nonlinear Dynamics (SINDy)** – identifies governing equations using sparse regression  

The dataset is generated numerically by simulating the system under multiple initial conditions.

The main objective is to evaluate whether these methods can successfully reconstruct system dynamics and to compare their performance in terms of accuracy and stability.

---

## 📂 Dataset

The dataset is synthetically generated by solving the nonlinear dynamical system:

- 100 different initial conditions  
- Time series data generated over a fixed time interval  

Numerical solutions are obtained using a **Runge-Kutta-based solver**.

## 📊 Results

The following results are obtained from the implemented methods:

- DMD fails to capture nonlinear system behavior and diverges over time  
- SINDy successfully reconstructs the system dynamics with high accuracy  
- SINDy achieves significantly lower RMSE compared to DMD  
- The nonlinear structure of the system is correctly identified by SINDy
